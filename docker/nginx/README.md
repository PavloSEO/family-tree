# Nginx для Family Tree

Архив **`nginx.zip`** с примерами для **Next.js + Strapi** в git не коммитится (см. **`.gitignore`**); при необходимости храните локально. Для Family Tree используется **упрощённый** вариант:

- **`Dockerfile`** — образ `nginx:1.27-alpine`, без компиляции динамических модулей.
- **`family-tree.conf`** — один `server` на порту 80, проксирование всего трафика на сервис **`family-tree:3000`** (Hono + SPA).

Запуск: из корня **`docker compose up -d --build`**. Сайт с хоста: **http://localhost:8080**. Порт **3000** у приложения по умолчанию только внутри сети Docker; чтобы открыть Hono с хоста на 3000, добавьте **`ports: ["3000:3000"]`** у сервиса **`family-tree`** в compose.
